%%%%
flame cloud

一团猛烈燃烧的怒火。如果没有完全的火焰抗性，
那么在火中的每一秒都会受到严重伤害。

穿过火云的远程冰霜攻击会降低射程，
同时驱散云层。

如果在水边释放，高温会蒸腾出大量蒸汽。
%%%%
noxious fumes cloud

一团呛人又有毒的云。
需要呼吸的生物如果没有毒抗性，一直站在毒云中，可能会导致混乱状态。
然而，经验丰富的生物不太会受此影响。
%%%%

freezing vapour cloud

一团冰冷的云。没有完全冰冻抗性的生物在里面会受到伤害。

穿过冰云的远程火焰攻击会降低射程，
消失弥散在冰云之中。
%%%%
poison gas cloud

一个充满致命毒气的云。
没有毒抗性的生物如果在里面，
会中毒并受到直接伤害。

%%%%
translocational energy cloud

这是某个最近来过这里的生物的残留物。但它已经不在了，被放逐到附近，
或穿过楼层，或者脱离现实，直到深渊。
一些巫师说，这是宇宙产生的一种“过敏反应”，
是对魔法以不可能的方式扭曲空间的反应；宇宙的喷嚏。
也许形而上学意义上是恶心的，但在其他方面无害。
%%%%
spreading flames cloud

熊熊的森林大火。
任何不能完全免疫火焰的生物都会因在火焰中度过而受到伤害。
火焰会间歇地爆发，并让周围的环境充满更小的火。
不幸的是，附近的树木，常常自己会变成更多的火种。

穿过火云的远程冰霜攻击会降低射程，同时驱散云层。

如果放置在水域附近或上方，热量使得附近产生蒸汽。
%%%%
steam cloud

灼热的蒸汽云。在其中的生物会轻微受伤，火抗属性会抵消伤害。
%%%%
ink cloud

墨云，里面完全是水。
%%%%
calcifying dust cloud

石化云。在云中呆任何时长的生物都会发现自己在变成石头，
尽管更有经验的生物也许能够抵抗这种影响。
%%%%
blessed fire cloud

圣火云，充满了神圣的力量。那些服侍善神的将不受影响。
其他生物会受伤，而那些亡灵和恶魔将受到很大的伤害。

thin mist cloud

薄雾云。营造氛围的，但无害。
%%%%
seething chaos cloud

暴怒混乱云。身处云中会受到各种不可预测的影响，
从隐形到狂暴，到完全麻痹。
%%%%
rain cloud

极度集中的暴风雨。它可能形成浅水池，或临时加深现有水池。
完全由火形成的生物不愿在雨中呆太久。
%%%%
mutagenic fog cloud

一团纯净、无结构的魔法，凝结成一团闪烁的薄雾，本质上是有毒的魔法废料。
冒险进入其中的生物会发现自己逐渐受到魔法辐射的污染，
当消散时，足够多的辐射会造成负面变异。
%%%%
magical condensation cloud

某种高度集中的魔法能量被唤醒，本身是完全无害的。
%%%%
raging winds cloud

猛烈的龙卷风。它会对任何身处其中的人造成严重伤害。
%%%%
sparse dust cloud

薄灰灰，营造氛围的，但无害。
%%%%
spectral mist cloud

幽灵般的薄雾云。它对除亡灵以外的所有生物造成轻微伤害。
随着时间的推移，危险却短暂的亡灵生物将从雾中聚集。
%%%%
acidic fog cloud

腐蚀肉身的酸液云雾，对身处其中的生物造成伤害和酸蚀。
抗酸属性可以带来免疫效果。
%%%%
thunder cloud

黑暗的暴风云。它会间歇地用闪电击打里面的人，
对那些没有电抗性的造成巨大伤害。
%%%%
negative energy cloud

巫术能量云。身处其中的生物会发现自己的生命能量在衰竭，
尽管对负能量的防护可以减轻其效果。
%%%%
white fluffiness cloud

白色蓬松的云。实际上用作枕头还不够结实，但也出奇地平和。
%%%%
salt cloud

一团急速旋转的尘和盐。它会刺入任何被捕获的生物的眼睛和皮肤，
尽管它最终是无害的。
%%%%
golden dust cloud

闪闪发光的金色尘云。身处其中，门派的追随者会在战斗中获得了巨大的力量。
